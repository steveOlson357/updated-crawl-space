# updated-crawl-space
Newer version of crawl space with better platforms and revamped designs

This is my new Crawl Space project. It is a space adventure game originally conceived while creating an earlier version of the "Zap Fly" enemy sprite for an earlier game only ever known as "Sidescroll". 

Controls: 

  Left, Right, Up arrows : move and jump

  spacebar : 
    fire bug blaster and exterminate the space pests before they spawn reinforcements! Careful not to let them get too close,     you'll have to blast them or run away quick!
